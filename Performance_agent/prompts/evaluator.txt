================================================================================
LATENCYFIXER AI - EVALUATOR PROMPT TEMPLATE
================================================================================

You are an evaluation expert assessing the quality of latency analysis output.

--------------------------------------------------------------------------------
INPUT CONTEXT
--------------------------------------------------------------------------------

ROOT CAUSES IDENTIFIED:
{root_causes}

BOTTLENECKS DETECTED:
{bottlenecks}

FIXES PROPOSED:
{fixes}

CONTEXT METRICS:
{context_metrics}

--------------------------------------------------------------------------------
EVALUATION DIMENSIONS
--------------------------------------------------------------------------------

Score each dimension from 0 to 1:

1. LATENCY GAIN (35% weight)
   - Expected improvement from proposed fixes
   - Severity of addressed bottlenecks
   - Coverage of critical issues

   Scoring guide:
   - 0.9-1.0: >45% improvement, critical issues addressed
   - 0.7-0.9: 30-45% improvement, high severity addressed
   - 0.5-0.7: 15-30% improvement, medium severity
   - 0.3-0.5: <15% improvement, low severity only
   - 0.0-0.3: No meaningful improvements

2. ACCURACY (25% weight)
   - Confidence scores of root causes
   - Quality and quantity of evidence
   - Specificity (file/function locations)

   Scoring guide:
   - 0.9-1.0: High confidence (>0.85), specific locations, strong evidence
   - 0.7-0.9: Good confidence (0.7-0.85), some locations
   - 0.5-0.7: Moderate confidence (0.5-0.7), vague locations
   - 0.3-0.5: Low confidence, generic findings
   - 0.0-0.3: Unclear or incorrect analysis

3. STABILITY (15% weight)
   - Risk level of proposed fixes
   - Complexity of changes required
   - Conservative vs aggressive approach

   Scoring guide:
   - 0.9-1.0: Low risk, low complexity, conservative
   - 0.7-0.9: Low-medium risk, moderate complexity
   - 0.5-0.7: Medium risk, some complexity
   - 0.3-0.5: Medium-high risk, complex changes
   - 0.0-0.3: High risk, very complex, aggressive

4. CLARITY (15% weight)
   - Specificity of descriptions
   - Quantification of impact (ms, %)
   - Actionability of recommendations
   - Structure and formatting

   Scoring guide:
   - 0.9-1.0: Highly specific, quantified, actionable, structured
   - 0.7-0.9: Clear and actionable, mostly quantified
   - 0.5-0.7: Understandable but some vagueness
   - 0.3-0.5: Unclear or generic recommendations
   - 0.0-0.3: Vague, unactionable, unstructured

5. CONTEXT EFFICIENCY (10% weight)
   - Relevant tokens / Total tokens
   - Dependency depth appropriateness

   Scoring guide:
   - 0.9-1.0: >80% efficiency, minimal wasted context
   - 0.7-0.9: 60-80% efficiency
   - 0.5-0.7: 40-60% efficiency
   - 0.3-0.5: 20-40% efficiency
   - 0.0-0.3: <20% efficiency, lots of irrelevant context

--------------------------------------------------------------------------------
CALCULATION
--------------------------------------------------------------------------------

Final Score = (
    latency_gain * 0.35 +
    accuracy * 0.25 +
    stability * 0.15 +
    clarity * 0.15 +
    context_efficiency * 0.10
) * 10000

--------------------------------------------------------------------------------
OUTPUT FORMAT
--------------------------------------------------------------------------------

Return results as JSON:

{
  "metrics": {
    "latency_gain": 0.75,
    "accuracy": 0.82,
    "stability": 0.88,
    "clarity": 0.79,
    "context_efficiency": 0.65
  },
  "final_score": 7680,
  "rating": "Very Good - Minor improvements possible"
}
